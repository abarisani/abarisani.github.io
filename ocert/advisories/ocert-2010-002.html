#2010-002 Joomla input sanitization errors (XSS)

Description:

Joomla, an open source content management system, suffers from a cross-site scripting (XSS) vulnerability.

Insufficient input sanitization on the parameters passed to pages related to administration settings leads to arbitrary javascript injection in the context of the user session, this could be potentially exploited to hijack the session of the Joomla administrator.

Affected version:

Joomla <= 1.5.19

Fixed version:

Joomla >= 1.5.20

Credit: vulnerability report and PoC received from Mesut Timur <mesut [at] mavitunasecurity [dot] com>.
